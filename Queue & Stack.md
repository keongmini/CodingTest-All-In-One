# Queue and Stack

## Queue(Deque)
시간 순서상 먼저 저장한 데이터가 먼저 출력되는 선입선출 형식으로 데이터를 저장하는 자료구조 enqueue / dequeue
  - Array List Based - Circular queue
  - **Linked List Based**
  - **FIFO**

```python
from collections import deque
queue = deque()
```


## Stack
시간 순서상 가장 최근에 추가한 데이터가 가장 먼저 나오는 후입선출 형식으로 데이터를 저장하는 자료구조 push / pop
  - **Array List Based**
  - Linked List Based
  - **LIFO**
